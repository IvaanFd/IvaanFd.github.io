FFit mobile app

This Final Degree Project focuses on the development of an application for mobile devices with Android operating system that allows users to manage their routines, set body weight goals, calculate possible weights to move and consult personal statistics.

Although there are currently numerous applications with this or similar functionality, there is no application that brings them all together. This work tries to explain the process carried out for the development and implementation of both the client side and the server side. The client side corresponds to the application interface with which users interact, while the server side is divided into two databases, one relational, persistent and managed with MySQL and the other nonpersistent, non-relational and managed with Redis, and two servers, one of them acting as an authentication server and the other as an application server.

In addition, it tries to explain the technologies used during the development, the tests carried out to check that the system works as expected and correctly will be discussed.

To develop the client side, Android Studio was used as the development environment and Kotlin as the programming language. For the server side, regarding the development of the two servers and their application programming interfaces (APIs), Express.js was used, a framework written in JavaScript and hosted within the Node.js execution environment. And with regard to the databases, two databases were used, a database that stores the information created and consulted by the two servers and another database hosted on the application server that acts as a cache memory.

Advantages of using FFit

  • Workout routine planning
  • Training statistics
  • Accurate exercise information
  • One-rep max calculator
  • One-rep max management
  • Body weight tracking
